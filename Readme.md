# Dolphin MPN

A Nintendo GameCube and Wii emulator built on the [Dolphin Emulator](https://github.com/dolphin-emu/dolphin), extended for Mario Party Netplay with seamless online sessions, built-in cheats, Netplay community tools, and traversal server optimizations.

Buildbots for Windows and Linux are powered by GitHub Actions' hosted runners.

## Supporting the Traversal Server

Online netplay relies on the Dolphin Traversal servers so players behind routers and complex firewalls can connect to each other. That server is purchased and operated by the upstream Dolphin Team, who currently funds it.

Dolphin Team does not accept donations for their services.
